<h1> challenge Conversor De Monedas </h1>

- Estado del proyecto terminado.

Este proyecto es un conversor de monedas que facilita la conversion de distintas monedas (USD, ARS, BRL, COP) entre si, con valores actualizados y facil de usar 
obtiene los valores reales mediante una API de ExchangeRate-API. 

Caracteristicas del conversor:

-Obtiene datos en tiempo real desde una API publica.

-Muestra los valores actualizados de las monedas:

USD (Dólar estadounidense)
ARS (Peso argentino)
BRL (Real brasileño)
COP (Peso colombiano)

-Menu interactivo con el usuario.

-Puede convertir todas las monedas disponibles ida y vuelta 

 USD → ARS / ARS → USD  
 USD → BRL / BRL → USD  
 USD → COP / COP → USD
 
-icorporacion de JSON por la libreria Gson

-codigo abierto a incorporar mas monedas
